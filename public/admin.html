How long do changes take to go live?

Redirect URLs /image-cache/...

Cached for 60 seconds. After you save a change here, visitors will start seeing the new image within about a minute. During the next 5 minutes, browsers may briefly serve the old image while fetching the new one in the background.

Final images (IMGIX)

Cached for 60 days by IMGIX. This doesn't affect rollout speed because each campaign uses a unique image URL. When you change the redirect target, the new URL points to a different image file entirely.

In short

After saving, all visitors will see the new image within ~1 minute, worst case ~6 minutes. No cache purging or deploys needed.
